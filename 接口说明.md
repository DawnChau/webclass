# User
## 用户登录获取token
- 方法
```
POST
```
- URL
```
/user/login
```
- 参数
```
{
	"username": "zhoufuxiao",
	"password": "123456"
}
```
- 是否需要携带Token
```
否
```
- 返回示例：
```json
{
    "msg": "用户登录成功",
    "data": "eyJhbGciOiJIUzUxMiJ9.eyJzdWIiOiJ6aG91ZnV4aWFvIiwiZXhwIjoxNTQ0NTM4NTcxfQ.h5q8K_vQcAjDDNTq3EbEGFVv4g8p4J4lyEhsi3AWN-fyajX-yhxopPh0rtvv8wmOJAyW-Gn-8CEWJeMjE827UQ"
}
```
- Case
```
"用户登录成功"
"用户不存在"
"用户密码错误"
"用户被封"
```

```

```

```
/admin/disable/{id}
```
- 参数
```
Integer id
```
- 是否需要携带Token
```

```

```json
{
```
